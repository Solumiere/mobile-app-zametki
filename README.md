# 🎯 Мои Цели — приложение целей, привычек и дневника

Мобильное приложение (PWA) для постановки целей, отметки их выполнения, трекинга привычек, ведения дневника и просмотра итогов месяца.

- Работает **офлайн** — данные хранятся прямо на телефоне (localStorage).
- Устанавливается на экран «Домой» как обычное приложение.
- Никакого сервера и регистрации не нужно.
- Чистый HTML/CSS/JS, без фреймворков.

## Содержимое

```
index.html             — интерфейс и стили
app.js                 — вся логика
manifest.webmanifest   — описание PWA
sw.js                  — офлайн-кэш (service worker)
icon.svg               — иконка приложения
```

## Как открыть на телефоне через GitHub Pages

Репозиторий: `Solumiere/mobile-app-zametki`.

> ⚠️ Сейчас репозиторий приватный. Бесплатный GitHub Pages работает только с публичным репозиторием. Сделай его публичным: **Settings → General → Danger Zone → Change repository visibility → Public** (либо используй тариф GitHub Pro).

1. Открой **Settings → Pages**.
2. В разделе *Branch* выбери `main` и папку `/ (root)`, нажми **Save**.
3. Через минуту появится ссылка вида `https://solumiere.github.io/mobile-app-zametki/`.
4. Открой эту ссылку на телефоне:
   - **iPhone (Safari):** Поделиться → «На экран Домой».
   - **Android (Chrome):** меню ⋮ → «Установить приложение».
5. Готово — иконка появится на экране, приложение работает офлайн.

## Проверить локально

```bash
python3 -m http.server 8080
# затем открой http://localhost:8080
```

## Функции

- 🏠 **Главная** — прогресс целей за месяц, лучший стрик, сводка.
- 🎯 **Цели** — добавление, категория, приоритет, срок, отметка выполнения, фильтры.
- 🔁 **Привычки** — ежедневный трекер с подсчётом стрика.
- 📓 **Дневник** — настроение, 3 хорошие штуки дня, рефлексия.
- ⚙️ **Настройки** — долгое нажатие на заголовок: имя, экспорт/импорт данных.
